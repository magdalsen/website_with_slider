# Website with slider

Project is simple website with image slider and contact form.

## Technology

- React + Vite + TypeScript
- React Router
- Yup
- React Hook Form
- Chakra UI
- Sass
- Pure React Carousel
- Prettier, Eslint

## Run

- Local: npm run dev
- Deploy: https://website-with-slider.vercel.app/
